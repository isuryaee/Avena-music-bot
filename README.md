# 🎵 Avena Music Bot

<h4 align="center">
🚀 A powerful, open-source Telegram music bot designed to provide a seamless listening experience.
</h4>

---

## 🔥 Features at a Glance

* 🎧 **Music Playback**: Stream high-quality music in Telegram groups/channels.
* ⏯ **Playback Controls**: Resume, pause, skip, and stop tracks easily.
* 🔍 **Search Engine**: Search for your favorite songs directly within the chat.
* 📜 **Playlist Support**: Manage queues and play multiple songs in a row.
* 🛠 **Admin Controls**: Manage bot access and permissions easily.
* ⚡ **Fast & Stable**: Optimized for low latency and high performance.

---

## ⚡ Quick Installation

Follow these steps to set up the bot on your local machine or server:

`git clone https://github.com/isuryaee/Avena-music-bot.git`
`cd Avena-music-bot`

1. Create a virtual environment: `python -m venv venv`
2. Activate: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`

---

## ⚙️ Configuration

Create a `.env` file in the root directory and add these variables:

- `API_ID`: Your API ID from my.telegram.org
- `API_HASH`: Your API Hash from my.telegram.org
- `BOT_TOKEN`: Your token from @BotFather
- `SESSION_STRING`: Your PyTgCalls session string
- `ADMIN_ID`: Your Telegram User ID

---

## 🚀 Running the Bot

Start the bot using:
`python main.py -p`
